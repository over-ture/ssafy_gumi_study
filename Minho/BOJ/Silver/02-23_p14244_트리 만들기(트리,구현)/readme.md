## 📌 문제 개요
$N 크기의 사이클이 없고 simple path가 1개인 무방향 연결 그래프('트리')에 대해서 리프노드가 M개일 때를 가정하고 N-1 개의 간선을 모두 출력하는 문제입니다.<br>
 ('차수(연결된 간선의 수)가 1개인 노드'를 리프노드라고 합니다.)


---

## 💡 해결 프로세스
 1. 0에서 1로 이어지는 리프노드 0번 노드를 하나 만듭니다.<br> 그 이후로 m-1 개의 노드가 남을때까지 일직선으로 노드를 연결합니다. 
 2. 일직선 으로 0번쨰 노드부터  ~ n-m 번째 노드까지 연결한 후, n-m 번째 노드에 n-(m-1) ~ n-(1) 번째 노드, 총 m-1개를 모두 연결합니다.
 3. 트리구조이므로 n개의 노드에 대한 간선은 n-1개 입니다.

---

## 💻 코드 구조 상세 (Core Logic)

🔍 뒤에서 m-1 개 만큼 n-m 번째 노드에 연결
```Java
    //n 개의 노드 n-1  개의 간선 -> 트리 리프노드는 m 개 
    //(0)->[1]->[2]->[3]....[n-m-1]->[n-m] ----(n - {m - 1} )()()(n-1)(n-1) / (m-1) 개   
    
    edge[0]= -1;
    edge[1]=  0;
       
    for(int node = n - (m -1) ;node < n ;++node) {
    	edge[node] = n-m;	   
    }
```
🔍 1~n-1 번째 노드까지 일직선으로 연결
```Java
    for(int i = 1 ; i<= n-m;++i) {
    	edge[i]= i-1;
    }
```
